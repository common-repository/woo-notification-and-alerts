== Description ==

[Recent Sales Notifications](https://woobewoo.com/plugins/woocommerce-popup-notifications/?utm_source=wordpress&utm_medium=description&utm_campaign=notifications "Recent Sales Notifications for WooCommerce") runs attractive recent sales popups to notify visitors of recently bought items. When visitors are aware that many other customers are buying from this store, they are more confident to make a purchase.

This plugin is useful for any WooCommerce shop and businesses of any size. Owners of WooCommerce stores can create custom popup notifications to promote any product, category and add promo codes and discounts, display the sale countdown. It creates a busy atmosphere for the visitors in the store, just like a busy retail store.

**[Any content to display](https://woobewoo.com/feature/any-content-text-products-categories-tags/?utm_source=wordpress&utm_medium=contenttodisplay&utm_campaign=notifications "Content")**
Promote any product, category, tag. Direct URL link allows visitors to click on the recent sales popups to view details in the page. 
Add numbers to display the random quantity of users or products, add users name and location, time, text.

Easily create attractive discount codes and use a countdown to create a feeling of urgency and demand for products.

Design a unique recent sales popup that matches your store in many ways: different templates, 9 positions on screen, background colour and opacity, overlay, borders, text and Close button style settings.

**[Responsive design](https://woobewoo.com/feature/responsive-design/?utm_source=wordpress&utm_medium=responsive-design&utm_campaign=notifications "Responsive design")**

== Installation ==

a) VIA FTP
Download a zip archive with Recent Sales Notifications for WooCommerce plugin
Extract files
Copy the folder with the plugin
Open via FTP folder \wp-content\plugins\
Paste plugin folder into here
Go to admin panel => Plugins => Recent Sales Notifications and press "Activate"

B) DIRECT UPLOAD VIA ADMIN AREA
Download a zip archive with plugin
Go to admin panel => Plugins => Add new
Choose the archive with a plugin and upload it
Wait until you get the notification installation went successfully
Press the "Activate" button

C) VIA WORDPRESS REPOSITORY
Navigate to admin panel => Plugins => Add new
Type \" Recent Sales Notifications for WooCommerce"\ in the search field
Choose our plugin from the list and click \"Install\"
Wait until you get the notification installation went successfully
Press the "Activate" button

== Frequently Asked Questions ==


= How to create a Recent Sales Notifications for my store? = 

We have all the necessary options in one plugin. It’s really easy to make the required settings and add Recent Sales Notifications into your online store.
To manage with an installation and plugin settings, check tutorial [Main Settings](https://woobewoo.com/documentation/woocommerce-popup-notification-main-settings/?utm_source=wordpress&utm_medium=main-settings&utm_campaign=notifications "Popup Notifications Main settings") and [Getting Started with Recent Sales Notifications for WooCommerce](https://woobewoo.com/documentation/woocommerce-popup-notifications-getting-started/ "Getting Started")

= How to add popup into your website
There are 2 different ways to add a popup to your site using code: table shortcode and table PHP code can be found on the left navigation menu if you click “Show All Notifications”
You can also find popup shortcode and PHP code if you open Notification to change settings. Simply choose a necessary code from the drop-down menu and copy the code. In order to insert the notification, you need:
1. Copy the shortcode.
2. Paste the shortcode into your post or page and Recent Sales Notifications will automatically create a notification on your page according to its settings
3. The PHP code can be inserted at any place of the page code. Besides, it can be inserted in the code of template of the theme – to display on all pages.

= How to change the notification design?

Recent Sales Notifications cames with lots of settings for every element. Navigate to the Design -> View Styles there you are able to change the popup position on-page, colour and transparency of background, borders styles, text fonts settings, close button design. More information [you can find here.](https://woobewoo.com/documentation/woocommerce-popup-notification-main-settings/#1/?utm_source=wordpress&utm_medium=ldesign&utm_campaign=notifications "Design")

= How to determine where and how to display notification?

To set the way to display the Popup on the website, move to the Rules to display tab.
Notification can be set to display according to the next parameters:

* Time on page – popup appears on page depend on the selected time and value.
* Time on site – popup appears on page depend on the time the user spent browsing the website.
* Distance scrolled – popup appears than user scrolling the page, depending on the value in px.
* Visitor inactive time – set the time of visitor inactivity to display a popup.
* The date – set the date to display a popup
* The current time – set the time to display a popup
* The current day – set the day to display a popup
* A visitor is new – popup will be displayed only for new visitors.
